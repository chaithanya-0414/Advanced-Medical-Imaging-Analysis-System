## How to Capture

1. Run the app:
   ```powershell
   streamlit run app_enhanced.py
   ```

2. Use Windows Snipping Tool:
   - Press `Win + Shift + S`
   - Select area to capture
   - Save to this folder

3. Name files as listed above for easy reference

## Tips

- **Resolution**: Capture at 1920x1080 or higher
- **Quality**: Use PNG format for clarity
- **Cropping**: Remove unnecessary browser chrome
- **Consistency**: Use the same sample image for all screenshots
- **Annotations**: Consider adding arrows/labels for key features
